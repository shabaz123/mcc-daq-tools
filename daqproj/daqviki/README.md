

Pre-requisites:

daqhats code must be installed: https://github.com/mccdaq/daqhats
wiringPi must be installed
I2C must be enabled on the Pi (raspi-config)
Create a folder called /home/pi/development/daqproj and place the code there.
The path of the code will be /home/pi/development/daqproj/daqviki

To build, type:
make all

Type:
sh ./install_service.sh
